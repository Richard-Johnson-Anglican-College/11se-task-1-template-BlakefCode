# 11SE Task 1 Submissions

This repository contains all student submissions for Task 1 - Hangman Game Project.

## Overview

Each student folder contains their individual implementation of the Hangman game as part of the Year 11 Software Engineering assessment task.

## Student Submissions

- **AZMEER** - Hangman game implementation
- **BLAKE** - Hangman game with web interface
- **HARSHAAN** - Hangman game with custom word bank
- **JOHAN** - Legacy hangman implementation
- **MOKSH** - Hangman game with custom words
- **PEP8** - Analysis report on code quality
- **SHAHIL** - Hangman game with real words dictionary
- **SREYANSH** - Hangman game implementation
- **TAREQ** - Hangman game with word bank

## Project Requirements

The Hangman game project required students to:
- Implement a fully functional Hangman game in Python
- Include a word bank or dictionary
- Handle user input and game logic
- Follow PEP8 coding standards
- Document their code appropriately

## How to Use

1. Navigate to a student's folder
2. Run the Python file (typically named `hangman.py`, `Hangman.py`, or similar)
3. Follow the on-screen instructions to play the game

Example:
```bash
cd AZMEER
python "Hangman_game_project - Azmeer.py"
```

## Assessment Criteria

Submissions were evaluated based on:
- Code functionality and correctness
- Code quality and PEP8 compliance
- Use of appropriate data structures
- Error handling and edge cases
- Documentation and comments
